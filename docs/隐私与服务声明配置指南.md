# 微信公众平台：服务内容声明 & 用户隐私保护指引

> 路径：**微信公众平台** → **设置** → **基本设置 / 服务内容声明**、**用户隐私保护指引**  
> 小程序名称示例：**台球天梯**（以你后台实际名称为准）  
> 请与 `miniprogram/pages/agreement/agreement.js` 内文案保持一致。

---

## 一、服务内容声明（简要说明，可整段粘贴）

```
本小程序为台球俱乐部线下服务工具，面向到店打球用户。主要功能：微信登录（需昵称，可选头像）、选择所在台球俱乐部（需位置授权，可选手动选择）、扫描本俱乐部球台二维码、对局记分、天梯积分与排行、积分商城兑换、个人战绩查询。用户须到店使用，数据由球房/平台运营方管理。
```

---

## 二、用户隐私保护指引（按信息类型填写）

在公众平台「用户隐私保护指引」中，对下列类型逐项添加（名称以后台下拉选项为准，选最接近的一项即可）。

### 1. 微信昵称

| 字段 | 填写内容 |
|------|----------|
| **使用目的** | 用于创建账号、在天梯排行榜、对局界面、桌台备战区展示用户身份，区分对战双方。 |
| **使用场景** | 用户点击「微信授权登录」或填写昵称登录时；进入天梯榜、对局页、个人中心时展示。 |
| **是否必要** | 是（拒绝提供将无法登录） |
| **收集方式** | 用户主动授权（微信头像昵称填写能力 / getUserProfile） |

### 2. 微信头像

| 字段 | 填写内容 |
|------|----------|
| **使用目的** | 用于个人中心、排行榜、对局界面展示用户头像，提升识别度。 |
| **使用场景** | 用户登录时授权选择头像；个人资料与排行列表展示。 |
| **是否必要** | 否（用户可不提供，使用默认头像） |
| **收集方式** | 用户主动授权（chooseAvatar / getUserProfile） |

### 3. 微信 openid（若后台有「微信账号信息」或「用户标识」）

| 字段 | 填写内容 |
|------|----------|
| **使用目的** | 用于识别用户身份、维持登录状态、关联对局与积分数据。 |
| **使用场景** | 用户登录时由微信接口返回，用于账号体系。 |
| **是否必要** | 是 |
| **收集方式** | 微信登录接口自动获取 |

### 4. 地理位置（位置信息）

| 字段 | 填写内容 |
|------|----------|
| **使用目的** | 用于匹配用户所在的台球俱乐部，在首页展示当前俱乐部名称与距离；校验用户仅能扫描所选俱乐部的球台二维码；距球房超过50米时提示确认。 |
| **使用场景** | 用户打开小程序进入首页时，在授权弹窗同意后获取一次当前位置；用户点击「重新定位」时。 |
| **是否必要** | 否（用户可拒绝定位，改用手动选择俱乐部） |
| **收集方式** | wx.getLocation（gcj02），仅在使用时获取，不后台持续定位 |

### 5. 对局/积分等业务数据（若需单独声明可选「其他」）

| 字段 | 填写内容 |
|------|----------|
| **使用目的** | 记录对局结果、积分变动、兑换记录，提供天梯排行与战绩查询。 |
| **使用场景** | 用户扫码对战、查看排行与个人中心时。 |
| **是否必要** | 是 |
| **收集方式** | 用户使用业务功能时产生 |

## 三、隐私保护指引（全文摘要，部分版本可粘贴）

```
我们收集：①微信openid用于登录；②您授权的昵称（必要）与头像（可选）用于展示；③经授权的位置信息用于匹配台球俱乐部与球台扫码校验，不持续定位；④对局与积分等业务数据。

我们用于：账号登录、俱乐部匹配、对战记分、排行与兑换、安全风控。不出售个人信息。数据存储于运营方服务器。您可拒绝定位并手动选俱乐部、退出登录或联系球房处理账号数据。

详见小程序内《隐私政策》。
```

## 五、小程序内配置核对清单

- [ ] `app.json` → `permission.scope.userLocation.desc` 与隐私指引一致  
- [ ] `requiredPrivateInfos` 含 `getLocation`  
- [ ] 登录页已勾选《用户服务协议》《隐私政策》  
- [ ] 首次调用 `getUserProfile` / `getLocation` 前会触发 `wx.requirePrivacyAuthorize`（基础库支持时）  
- [ ] 体验版可演示：首页俱乐部栏、登录授权、排行头像昵称
